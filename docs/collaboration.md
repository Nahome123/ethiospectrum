# Collaboration

Use `ETH-###` issue identifiers and branches such as `feature/ETH-012-secure-document-upload`, `fix/ETH-031-upload-timeout`, `chore/ETH-042-update-dependencies`, and `docs/ETH-050-security-model`. Keep pull requests small, document decisions, and complete security/accessibility/translation sections. Do not add real family data to issues, screenshots, logs, or fixtures.
